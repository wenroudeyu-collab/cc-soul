# cc-soul — Give Your AI a Soul

A cognitive architecture plugin for OpenClaw. 50 modules, 19,600+ lines, 123 features.

- **Long-term Memory** — 10,000+ memories, semantic search, auto-merge, contradiction detection, 3-tier decay, archival reactivation
- **Adaptive Personality** — 5 personas blend by context (engineer/friend/mentor/analyst/comforter), style drift detection
- **Emotional Awareness** — bidirectional mood transfer, frustration detection, 7-day emotional arc
- **Self-Evolution** — Bayesian hypothesis verification, root cause attribution, strategy replay, meta-learning
- **Autonomous Behavior** — dream mode, web research, competitive radar, proactive conversation
- **Knowledge System** — lorebook, epistemic calibration, knowledge graph, RAG ingestion
- **Self-Upgrade** — 5-agent pipeline, syntax check, observation period, auto-rollback

## Install

```bash
openclaw plugins install @cc-soul/openclaw
```

Zero configuration. Auto-detects your AI backend. All data stays local.

## 36 Toggleable Features

```
enable dream_mode    → turn on
disable web_rover   → turn off
features            → list all
stats               → personal dashboard
soul state          → energy/mood/alertness
```
